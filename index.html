LAPA 시선측정 프로토타입

리더스아이 LAPA(Language Ability & Process Analysis) 서비스의 주요 화면을 미리 체험해보실 수 있습니다. 실제 데이터 대신 더미 데이터가 사용됩니다.

🗓 2025년 6월 프로토타입 📄 5개 화면 🔒 내부 검토용
💡
이 페이지는 이화여대 담당자 분들의 사전 검토를 위한 프로토타입입니다. 모든 데이터는 더미이며, 실제 서비스 UI·기능은 변경될 수 있습니다. 의견이나 피드백은 담당자에게 전달해 주세요.
수험자 화면
🖥
키오스크
PSAT 시선측정 키오스크
수험생이 키오스크 앞에 앉아 시선측정을 진행하는 전체 흐름입니다.

포함 화면: 이름·집단 입력 → 글 선택(진단/훈련) → 시선 보정(5점) → 후보정 앵커 → 지문 읽기(좌 60%) + 객관식 풀이(우 40%) → 결과 확인 및 전송
로그인› 글 선택› 시선 보정› 측정› 결과

개별 측정
자유글 시선측정 (피검사자)
피검사자가 자유롭게 제공된 글을 읽는 동안 시선을 측정하는 화면입니다. 키오스크와 달리 문제 풀이 없이 자유 독서 방식으로 진행됩니다.

포함 화면: 안내 → 글 표시(전체 화면) → 측정 진행 표시 → 완료

분석 뷰어
시선 분석 결과 뷰어
연구자가 수집된 시선 데이터를 분석하는 화면입니다. 여러 참여자의 시선 경로를 동시에 비교할 수 있습니다.

주요 기능: 시선경로 재생(응시·비짓 모드) / Scarf 이동경로 다이어그램 / AOI 전이 매트릭스 / 참여자 간 유사도 비교(Levenshtein·LCS·Bigram)
